预警API文档

预警系统的API接口文档

API概览

预警API提供预警的创建、查询和处理功能。所有API请求都需要在Header中携带认证信息。

Base URL: https://api.paygateway.com/v1/risk/warnings
认证方式
Authorization: Bearer {your_api_key}
创建预警

创建一个新的预警记录。

POST /warnings
请求示例
{
  "source": "ETHOCA",
  "source_warning_id": "ETH202310201423156782",
  "order_no": "PY202310201423156782",
  "warning_type": "DISPUTE",
  "warning_level": "HIGH",
  "risk_score": 85,
  "warning_data": {
    "amount": 268.00,
    "currency": "USD",
    "customer_email": "customer@example.com"
  }
}
响应示例
{
  "code": 0,
  "message": "成功",
  "data": {
    "warning_id": "WARN202310201423156782",
    "status": "NEW",
    "created_at": "2023-10-20T14:23:15Z"
  }
}
查询预警列表

查询预警列表，支持分页和筛选。

GET /warnings
查询参数
参数 类型 说明
page integer 页码，从1开始
size integer 每页数量，默认20
status string 预警状态：NEW, PROCESSING, RESOLVED
warning_level string 预警级别：LOW, MEDIUM, HIGH, CRITICAL

Order Insight API文档

Order Insight系统的API接口文档

Order Insight API

Order Insight API提供订单风险分析和洞察功能。

Base URL: https://api.paygateway.com/v1/order-insight
提交订单分析

提交订单进行风险分析。

POST /analyze
请求示例
{
  "order_id": "ORDER123456",
  "amount": 268.00,
  "currency": "USD",
  "customer_id": "CUST001",
  "customer_email": "customer@example.com",
  "billing_address": {
    "country": "US",
    "state": "CA",
    "city": "Los Angeles"
  },
  "items": [
    {
      "name": "Product A",
      "price": 268.00,
      "quantity": 1
    }
  ]
}
响应示例
{
  "code": 0,
  "message": "成功",
  "data": {
    "risk_score": 65,
    "risk_level": "MEDIUM",
    "recommendation": "建议进行3DS验证",
    "analysis_time": "2023-10-20T14:23:30Z"
  }
}

Checkout API文档

支付收银台的API接口文档

Checkout API

Checkout API提供支付收银台集成功能。

Base URL: https://api.paygateway.com/v1/checkout
创建支付订单

创建一个新的支付订单。

POST /orders
请求示例
{
  "order_no": "ORDER123456",
  "amount": 268.00,
  "currency": "USD",
  "description": "Product Purchase",
  "customer_id": "CUST001",
  "return_url": "https://your-domain.com/success",
  "cancel_url": "https://your-domain.com/cancel"
}
响应示例
{
  "code": 0,
  "message": "成功",
  "data": {
    "checkout_url": "https://checkout.paygateway.com/pay/ORDER123456",
    "order_id": "ORDER123456",
    "expires_at": "2023-10-20T15:23:15Z"
  }
}

Webhook API文档

Webhook回调接口文档

Webhook API

Webhook API用于接收支付网关的实时事件通知。

事件类型: 支付成功、支付失败、退款完成、预警通知等
Webhook验证

所有Webhook请求都会携带签名头，用于验证请求来源。

X-Pay-Signature: sha256=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
支付成功事件

支付成功时的Webhook通知示例。
